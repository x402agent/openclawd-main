# TailClawd Backup

Backup and restore utilities for TailClawd state and sessions.

## Overview

This directory contains backup tools for the TailClawd web interface, including session state persistence and data export.

## Usage

```bash
# Create backup
./backup.sh

# Restore from backup
./restore.sh backup-2024-01-01.tar.gz
```
