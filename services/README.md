# Services

Backend services for the OpenClawd ecosystem.

## Overview

This directory contains backend service implementations including gateway, bridge, and integration services.

## Components

### Gateway

TCP/IP gateway services for node communication and bridging.

### Bridge

Protocol bridge implementations for cross-chain and cross-platform operations.

### Monitoring

System monitoring and health check services.

See individual service directories for specific implementations and setup instructions.
