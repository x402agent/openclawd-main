# npm Packages

npm package installers for the OpenClawd ecosystem.

## Packages

### clawdbot-installer

Installation scripts for the ClawdBot CLI tool.

### solanaos-installer

Installation scripts for the SolanaOS CLI tool.

## Usage

```bash
# Install ClawdBot
npm i -g @clawd/cli

# Install SolanaOS
npm i -g solanaos-cli
```
