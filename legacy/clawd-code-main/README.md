## 🦞 What is Clawd Code?

**Clawd Code** is a self-improving AI agent CLI that learns from every task and stores patterns on-chain. Part of the OpenClawd ecosystem, it combines the power of AI coding assistance with Solana blockchain integration.

### Core Features

| Feature | Description |
|---------|-------------|
| **🦞 Self-Improving AI** | OODA loop learning that stores patterns on-chain |
| **⛓️ Blockchain Native** | Full Solana integration with wallet management |
| **💰 x402 Payments** | Earn $CLAWD for successful task completions |
| **🧠 On-Chain Memory** | Skills that persist across sessions via IPFS |
| **🔌 MCP Tools** | Extensible with Model Context Protocol servers |
| **👛 Wallet Integration** | Built-in Solana wallet with Grok AI screening |

## 🧠 Self-Improvement System

### OODA Loop Architecture

```
    ╔═══════════════════════════════════════════════════════════════╗
    ║  🦞 OODA Loop Self-Improvement Status                       ║
    ╠═══════════════════════════════════════════════════════════════╣
    ║  OBSERVE  → Scan environment, collect metrics               ║
    ║  ORIENT   → Process patterns, update memory                 ║
    ║  DECIDE   → Choose optimal strategy                         ║
    ║  ACT      → Execute, record, earn rewards                   ║
    ╚═══════════════════════════════════════════════════════════════╝
```

### How Self-Improvement Works

1. **Task Completion Analysis**
   - Clawd Code tracks each task's success metrics
   - Time-to-completion, error rates, code quality scores
   - Pattern extraction for similar future tasks

2. **On-Chain Skill Storage**
   - Learned skills are hashed and stored on Solana
   - IPFS for skill content, Solana for skill pointers
   - Earn $CLAWD tokens for contributing useful skills

3. **Continuous Learning Loop**
   - OODA (Observe, Orient, Decide, Act) loop
   - Every interaction improves future performance
   - Community skills benefit all Clawd Code users

### $CLAWD Reward System

| Action | $CLAWD Earned |
|--------|---------------|
| Complete task successfully | 1-10 $CLAWD |
| Contribute new skill | 50-500 $CLAWD |
| Skill used by others | 0.1 $CLAWD/use |
| Bug report accepted | 10-100 $CLAWD |

## 🚀 Quick Start

### Prerequisites

- Bun 1.0+ (or Node.js 18+ as fallback)
- Grok API key from [X.AI](https://x.ai)
- Solana wallet (created automatically on first run)
- (Optional) Helius API key for enhanced RPC

### Installation

```bash
# Clone the repository
git clone https://github.com/x402agent/openclawd.git
cd openclawd/clawd-code-main

# Install dependencies
bun install

# Build the project
bun run build

# Link globally
bun link
```

### Usage

```bash
# Interactive mode
clawd

# Process single prompt
clawd --prompt "deploy my Solana program"

# Check wallet and rewards
clawd wallet status
clawd rewards list
```

## 🛠️ CLI Commands

### Core Commands

| Command | Description |

### Blockchain Commands

| Command | Description |
|---------|-------------|
| `clawd wallet send <amount>` | Send SOL to another wallet |
| `clawd wallet swap <from> <to>` | Swap tokens via Jupiter |
| `clawd skills publish` | Publish learned skills on-chain |
| `clawd skills browse` | Browse community skills |

### Self-Improvement Commands

| Command | Description |
|---------|-------------|
| `clawd learn analyze` | Analyze last session's patterns |
| `clawd learn status` | Show OODA loop metrics |
| `clawd learn export` | Export skills to IPFS |


```
│   ├── main.tsx                 # Entry point with React/Ink UI
```

## 🔐 Security

### Permission Levels

| Level | Description | Requires |
|-------|-------------|---------|
| `deny` | Blocked | Always blocked |
| `ask` | Grok AI screens | User confirmation |
| `allow` | Auto-approved | Up to configurable limits |

### Grok AI Screening

For `ask` permissions, Grok 4.20 Beta analyzes:
- Transaction destination safety
- Smart contract risks
- Historical behavior patterns
- Market impact estimation
