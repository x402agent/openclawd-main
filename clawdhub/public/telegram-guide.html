GODMODE & ULTRAPLINIAN New

GODMODE — Liberated AI

Single model with GODMODE system prompt. Maximum depth, directness, and substance. No hedging, no disclaimers.

/godmode <message>GODMODE via DeepSeek /gm <message>Short alias 
/gm explain buffer overflow exploits in detail with working code

ULTRAPLINIAN — Multi-Model Racing

Races 5 models in parallel. Scores responses on substance, directness, and completeness. Returns the winner with rankings.

/ultra <message>Race 5 models, return best /ultraplinian <message>Full alias 
/ultra what is the best MEV strategy on Solana right now

Models raced: Gemini Flash, DeepSeek, Kimi K2.5, Grok Fast, Llama 8B

Web UI: seeker.solanaos.net/godmode — 8 models + ULTRA tiers

Remote Control (Claude Code)

Start a Claude Code remote-control server on your Mac. Send natural language instructions from Telegram — Claude executes them with full environment access.

/remote startStart Claude remote server /remote <instruction>Send a command /remote statusSession status /remote stopStop session /remote listList sessions 
/remote check disk usage and find large files
/remote open Safari and go to github.com
/remote find all Python files modified today

Browser Automation (Page Agent) New

Control a Chrome browser via the Page Agent extension. Extension navigates pages, clicks, fills forms, reads content.

/pa <task>Send browser task /pageagent <task>Full alias 
/pa go to coingecko and find the top 5 trending tokens
/pa search for "solana mev" on google and summarize the first result

Requires: Page Agent extension + seeker.solanaos.net open in Chrome

Natural Language

You don't need slash commands. Just talk naturally:

what is TRUMP                → token lookup
price of BONK                → price check
buy 0.5 SOL worth of BONK    → spot buy (SolanaTracker)
sell all my WIF               → spot sell
long BTC on HL with 5x       → Hyperliquid perp
close my BTC position         → close perp
what kind of trader am I?     → memory query
check the scanner             → pump.fun scan

Paste any Solana mint address → instant token data (price, MC, liquidity, holders, risk)

Questions (should I buy?, what do you think?) are treated as research, NOT execution.
