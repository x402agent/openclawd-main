# Telegram Bots

Telegram bot integrations for the OpenClawd ecosystem.

## Overview

This directory contains Telegram bot implementations that connect AI agents to the Telegram messaging platform.

## Available Bots

See individual bot directories for specific implementations.

## Integration

Bots communicate with the OpenClawd agent runtime via:
- **MCP Protocol** — For tool execution
- **A2A Protocol** — For inter-agent communication
- **ClawdRouter API** — For unified agent dispatch
