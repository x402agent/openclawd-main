# рҹҰһ mawd

> Lobster-powered CLI with stunning Unicode braille animation startup

A beautiful terminal experience with animated braille art, multi-phase boot sequences, and an interactive REPL вҖ” all lobster-themed.

## Install

```bash
npm install -g mawd
```

## Usage

```bash
mawd
```

That's it. Type `mawd` and watch the lobster materialize.

## What happens when you run `mawd`

1. **рҹҢҠ Particle Convergence** вҖ” Scattered braille dots converge into a lobster shape across 6 animation frames
2. **гҖ°пёҸ Water Wave** вҖ” Animated wave separator ripples across the screen
3. **рҹ”Ө Logo Typewriter** вҖ” The MAWD block-letter logo reveals line-by-line with a redвҶ’coralвҶ’amberвҶ’gold gradient
4. **рҹҰһ Claw Snap** вҖ” The lobster's claws animate in an idle snapping loop
5. **вҡЎ Boot Sequence** вҖ” 6 unique Unicode spinners cycle through boot messages using `unicode-animations`
6. **вң… Ready Prompt** вҖ” Drops you into an interactive REPL with the `рҹҰһ mawd вқҜ` prompt

## REPL Commands

| Command | Description |
|---------|-------------|
| `/help` | Show all available commands |
| `/predict <token>` | Get a price prediction |
| `/status` | System & network status |
| `/markets` | Browse prediction markets |
| `/portfolio` | View your positions |
| `/quit` | Exit mawd |

## Spinners

Uses [unicode-animations](https://www.npmjs.com/package/unicode-animations) for the boot sequence:
- `braille` вҖ” Classic braille spinner
- `scan` вҖ” Grid scan animation
- `helix` вҖ” DNA helix pattern
- `cascade` вҖ” Cascading dots
- `dna` вҖ” DNA strand
- `orbit` вҖ” Orbital motion

## Requirements

- Node.js вүҘ 18
- A terminal that supports Unicode & ANSI colors (most modern terminals)
