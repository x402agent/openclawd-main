What's Included

1

Tailscale + Tailnet

Installs Tailscale, logs you in, and sets up tailscale serve on port 3110 so your claw is accessible from any device on your tailnet.

2

solana-clawd CLI

Global npm package with OODA loop trading, 31 MCP tools, Go + TypeScript agent framework. Run from anywhere on your machine.

3

TailClawd Web UI

Browser-accessible Clawd Code via your private tailnet. 14 streaming endpoints, real-time SSE, live token counter, cost tracking, model selector.

4

@openclawd/wallet

Privy-powered embedded Solana wallet with Grok 4.20 Beta as the AI reasoning layer. Agent swap, permissions, on-chain settlement.

5

@solana-clawd/x402

Agent-to-agent USDC payment protocol via x402. Monetize MCP servers, HTTP APIs, and tool calls with one-line integration.

6

clawd-profile CLI

Profile manager inspired by Hermes — each profile is an isolated agent home with its own config, CLAW.md personality, memories, sessions, skills.

Profiles — Running Multiple Agents

Every agent needs a home. A profile is an isolated agent home directory with its own config, secrets, CLAW.md personality, memories, sessions, skills, and gateway state.

clawd-profile — profile manager
$ clawd profile create default ◉ created profile default at ~/.openclawd/profiles/default ◉ alias installed: ~/.local/bin/default ∎ edit ~/.openclawd/profiles/default/CLAW.md to shape this claw's personality $ clawd profile create degen-trader ◉ created profile degen-trader at ~/.openclawd/profiles/degen-trader ◉ alias installed: ~/.local/bin/degen-trader ∎ edit ~/.openclawd/profiles/degen-trader/CLAW.md to shape this claw's personality $ clawd profile list openclawd profiles (~/.openclawd/profiles) ● default (active) ○ degen-trader $ clawd profile use degen-trader ◉ active profile is now degen-trader $ clawd profile clone default arb-hunter ◉ cloned default → arb-hunter $ degen-trader "buy 0.1 SOL of $PEPE" 🦞 launching degen-trader profile...
CLAW.md

The Soul of the Claw

Each profile ships with a CLAW.md — personality charter defining identity, operating style, hard limits, and domain focus. Replace Hermes' SOUL.md with OpenClawd's lobster-themed equivalent.

MULTI-AGENT

Honcho Integration

Run multiple profiles in parallel with Honcho. Each service gets its own profile, own config, and own memory bank — zero state collision.
