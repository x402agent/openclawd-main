# Examples

Example code and demonstrations for the OpenClawd ecosystem.

## Available Examples

### blockchain-buddies-demo.ts

Blockchain integration demo showing wallet monitoring and transaction tracking.

### listen-wallet.ts

Wallet event listener example for monitoring on-chain activity.

### ooda-loop.ts

OODA (Observe-Orient-Decide-Act) trading loop demonstration.

### x402-solana.ts

x402 payment protocol integration example for Solana.

## Usage

```bash
# Run an example
npx tsx examples/blockchain-buddies-demo.ts
```
