# Chess

SolanaOS Chess hub and related chess components.

## Overview

This directory contains the chess integration for the SolanaOS ecosystem, including:
- Embedded live play interface
- Wallet-signed match archive
- Integration with SolanaOS Hub

## Projects

### chess-mcp-main

MCP server for chess-related operations.

### chess-web-api

Web API for chess game management and archival.

## Setup

```bash
npm install
npm run dev
```

## Features

- Live chess gameplay
- Wallet-based match signing
- Game archival on Solana
